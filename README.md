# Where Is My Book - SIFT Detector

Computer Vision project to detect a target book using SIFT feature matching.

## Project structure
- data/ : dataset images
- docs/ : proposal and reports
- results/ : output images/results
- src/ : helper source code
- main.py : main execution file
- requirements.txt : dependencies

## Target object
Red Victoria Memorial book.

## Method
- SIFT keypoint detection
- Descriptor matching
- Homography / localization of target book

## Dataset
- Reference images
- Positive images
- Negative images
